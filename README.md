# 🍕 Pizza Sales Analysis Dashboard

## 📌 Project Overview
This project focuses on analyzing pizza sales data using SQL and creating an interactive dashboard in Excel/Power BI to identify sales trends, customer behavior, and business insights.

---

## 🎯 Objectives
- Analyze sales performance
- Identify peak order times
- Find best and worst-selling pizzas
- Understand customer ordering patterns
- Build an interactive dashboard for visualization

---

## 🛠️ Tools & Technologies
- SQL
- Excel
- Power BI
- Data Cleaning
- Data Visualization

---

## 📊 Key KPIs
- Total Revenue
- Total Orders
- Average Order Value
- Total Pizzas Sold
- Sales by Category
- Sales by Size
- Daily & Hourly Trends

---

## 📈 Key Insights
- Friday had the highest orders
- Large pizzas generated maximum sales
- Peak order hours were 12 PM – 1 PM
- Classic category contributed the highest revenue
